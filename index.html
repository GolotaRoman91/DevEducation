Linked List

Linked list results are updated in real time, commands should be entered into the console. List of commands:

- addValue('some value') - to add one value

- addSomeValue(['value', 'one more value', 'and another more value', ...]) - to add some values in array

- getValueById(id) - to get value by id (result in console)

- getIdByValue('value') - to get element by value name (result in console)

- deleteValue('value') - to delete by value name

- deleteLastElem() - to delete last element

- deleteByID('id') - to delete value by id

- linkedList.'value name'.next() - to see linked element with this value (result in console)

- showHead() - to see head element (result in console)

- showTail() - to see tail element (result in console)

- getClear() - to clear Linked List
